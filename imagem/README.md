# Pasta Para Estudo de Manipulação de imagens com Python

## Biblioteca pillow

### Importando

```from PIL import Image```

### Usabilidade

#### Abrindo

```imagem = Image.open('caminho_arquivo.extensão')```

#### Salvando

```imagem.save('caminho_arquivo.extensão') ```
